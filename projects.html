SCCM Console Self-Elevation Script

PowerShell SCCM Windows Security Error Handling

Challenge: IT administrators needed seamless privilege escalation for SCCM Console operations without manual UAC prompts disrupting workflow.

Solution: Developed a PowerShell script that automatically detects current privilege level, performs self-elevation when needed, and includes robust error handling for common elevation failures (error code -2147024894). The script features session testing capabilities and maintains context across privilege boundaries.

Impact: Reduced manual intervention by 90% and eliminated workflow interruptions for system administrators managing enterprise deployments.

Cross-User Registry Manager

PowerShell Registry User Profiles PSADT

Challenge: Need to apply registry modifications across all user profiles (HKCU) on enterprise workstations, including users who weren't currently logged in.

Solution: Created a comprehensive PowerShell function using PSADT-style architecture that enumerates all user profiles, loads registry hives, applies changes systematically, and creates Active Setup entries for future users. Includes validation and rollback capabilities.

Impact: Enabled consistent application settings deployment across enterprise environment with 100% user coverage, regardless of login state.

Application Packaging Automation

MSI InstallShield SCCM Intune

JSON-Powered Configuration Manager

PowerShell JSON Configuration Management Automation

Challenge: Configuration data scattered across multiple files made script maintenance difficult and error-prone for complex deployment scenarios.

Solution: Created a PowerShell function that reads variables from JSON configuration files at script initialization, enabling centralized configuration management and easy parameter modification without script changes.

Impact: Simplified configuration management and reduced deployment errors by centralizing all variable data in easily maintainable JSON format.
